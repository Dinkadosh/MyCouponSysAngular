My name is Michael Zhebov and this is my project.

On this site you can register as a company or customer. The company create discount coupons, and customer buy them. Companies can add, edit and delete coupons. Customers can buy coupons and use them. Admin can view the lists of customers and the company, modify and delete them. It can also create new users. Everyone can change their profile.

On this project I used the following technologies:

  • Spring Boot
  • Spring JPA
  • Spring Web
  • Spring Security
  • JWT Token
  • PostgreSQL
  • Heroku
  • Angular 8
  • Angular Material
  • Angular Notifier
  • Bootstrap 4
