My name is Michael Zhebov and this is my project.

On this site you can register as a company or client. The company creates discount coupons, and customers buy them. Companies can edit and delete coupons. Both clients and companies can edit the profile.

On this project I used the following technologies:

  • Spring Boot
  • Spring JPA
  • Spring Web
  • Spring Security
  • JWT Token
  • PostgreSQL
  • Heroku
  • Angular
  • Angular Material
  • Bootstrap
